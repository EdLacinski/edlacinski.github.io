# Social preview image

The 1200 × 630 `eclipse-hub-preview.png` in this folder is the default Open Graph and social sharing image.

- Keep the design consistent with the black-and-green Eclipse Hub branding.
- Include the site name: “Edward Lacinski | Eclipse Hub”.
- Avoid tiny text that will be unreadable in LinkedIn, X, or Discord previews.
- Keep important content away from the outer edges where previews may crop it.

The site metadata points to `/images/social/eclipse-hub-preview.png`. Keep that filename when replacing the artwork, or update the metadata in `index.html` at the same time.
