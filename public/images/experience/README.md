# Experience images

Drop experience or job photos in this folder.

- Recommended formats: `.jpg`, `.png`, or `.webp`
- Recommended size: square images at least 600 × 600 pixels
- Use simple filenames such as:
  - `regal-entertainment.jpg`
  - `interboro-tech.jpg`
  - `interboro-data.jpg`
  - `eclipse-ed.jpg`
- Update the matching entry in `src/data/experience.js` if you use a different filename.
- Public image paths begin at the site root. Example: `/images/experience/regal-entertainment.jpg`

Until an image is added, the website automatically displays a styled initials placeholder.
