# Site icons

The included `favicon.svg` provides a lightweight browser icon. These additional production assets can be added later:

- `favicon.ico`
- `icon-192.png`
- `icon-512.png`
- `apple-touch-icon.png`

Use the black-and-green Eclipse Hub branding, keep the central mark simple, and check legibility at small sizes before referencing new files in `index.html`.
